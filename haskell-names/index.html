haskell-names-0.1: Resolving Haskell module dependencies.

haskell-names-0.1: Resolving Haskell module dependencies.

This is an addon to Haskell-Source with Extensions (HSE, haskell-src-exts). It handles additional tasks on top of parsing:

  • Chasing module dependencies to find all used modules.
  • Resolving all names in the modules and giving them fully qualified names.
  • Using symbol fixities to do the final parsing.
